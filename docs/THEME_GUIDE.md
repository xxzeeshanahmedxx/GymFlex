# Theme Guide

Main styling is in:

```txt
apps/storefront/src/index.css
apps/admin/src/index.css
```

Replace:

- CSS variables
- brand colors
- heading font feel
- hero/banner visuals
- product card shape
- button style
- collection banner style

Keep:

- readable mobile layout
- compact footer
- product image skeletons behind images
- 44px touch targets
- admin no-store behavior

Do not:

- add heavy animation libraries without reason
- hide images behind loading state
- make buttons awkward full-width on mobile
